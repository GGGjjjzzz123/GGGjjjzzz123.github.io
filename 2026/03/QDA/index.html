    论文：优化算法的量子动力学探讨

    1 优化算法的量子动力学理论

    将优化问题中的目标函数f(x)视为薛定谔方程中的势能，就是优化算法的量子动力学方程，这一方程转化为了势能f(x)下的约束态量子问题。得到的对应关系：波函数模方的演化过程=解的概率演化过程，目标函数的全局最小值=基态波函数模方问题

    （薛定谔方程：以电子运动的能量为约束条件，通过这种能量的约束去描述电子绕原子核周围产生的运动，解这个方程得到一系列波函数）

    4）目标函数在极值点的二阶Taylor近似与对应的迭代操作

    一整段的因果都不懂

    3 基本迭代过程的性能及波函数模方

    实验原理实验过程不懂，整个实验都不懂，这个实验是干什么的

    双阱函数有三个参数，修改那三个参数观察在不同的双阱函数上，算法的效率

    问题：函数的难度取决于什么

    l对函数图像的影响

    l=1.0,k=0.1,h=1.0

    l, k, h = 5.0, 0.1, 1.0

    k对函数图像的影响

    l, k, h = 1.0, 0.1, 1.0

    l, k, h = 1.0, 1.0, 1.0

    h对函数图像的影响

    l, k, h = 1.0, 0.1, 1.0

    l, k, h = 1.0, 0.1, 10.0

    调试l

    l, k, h = 5.0, 0.1, 1.0

    l, k, h = 0.01, 0.1, 1.0


    数据整理

    发现：

    1.l的值越小，算法的迭代次数越大，效率就越低，相对误差的数量级就越大，误差也就越大

    2.l的值越大，迭代次数越少（最终稳定在七八十左右），效率高，相对误差小

    3.在图像上看，l值越小，k对函数图像的影响就越小

    4.最小值的绝对值随着l的增大而增大

    调试k

    l, k, h = 5.0, 0.01, 1.0


    l, k, h = 5.0, 2.0, 1.0

    l, k, h = 5.0, 6.0, 1.0

    数据整理

    发现：

    1.当k值越小，迭代次数越大，效率低

    2.k值越大，迭代次数越小（稳定在80左右）

    3.相对误差稳定，正确率高

    4.最小值随着k的值增大而增大

    调试h

    l, k, h = 5.0, 0.1, 1.0

    l, k, h = 5.0, 0.1, 500


    数据整理

    发现：
    1.h的值越小，迭代次数越小，相对误差小

    2.h值越大，迭代次数越小，相对误差变化不大

    3.最小值不随h值变化
