Numpy矩阵运算

numpy为我们提供了一个特殊的数组对象，它可以对数组中的数据进行非常高效的运算

python是一个很慢的语言，但是numpy之所以能这么快是应为，它底层是用C语言实现的目标代码

numpy的核心：对于任何要运算的数据，需要预先将他们表示成numpy数组的形式——向量化、并行化

numpy中所有的运算都是围绕着数组进行的，因此在计算前我们需要将计算的数据表示成数组的形式

使用步骤：

  1. 导入numpy这个库并起个别名np
  2. 使用np.array([1,2,3,4,5])创建一个数组，括号中是初始化的数据

也可以用np.zeros((3,2))创建一个全零的数组，括号里传入的参数是数组的尺寸

np.ones((2,4))创建一个全一的数组

np.arange(3,7)创建一个递减的数列等间距分布的数

np.linspace(0,1,5)创建一个介于某个区间

np.random.rand(2,4)创建一个随机数组

  1. a.shape 可以用来获得数组的尺寸
  2. a.dtype 在Numpy中数组默认的元素类型为float64，可以在创建np数组时指定元素类型。对于现有的数组可以通过a.astype(int)来转换数据类型。

矩阵的基本运算

  1. 两个尺寸相同的数组可以直接进行四则运算，将数组同位置的元素进行加减乘除。
  2. 两个尺寸不同的数组也可以进行运算，运算之前需要广播，使得两个矩阵的尺寸相同。
  • np.dot(a,b) 将两个向量进行点乘运算
  • a @ b 矩阵的乘法（等同于np.matmul()函数）
  • np.array(a) 对所有数依次求平方根
  • np.sin(a) np.cos(a) 进行三角函数运算
  • np.log(a) 进行对数运算
  • np.power(a,2) 进行指数运算
  • a*5 单独与一个数进行运算，分别计算各个元素与这个数的乘积 —> 广播
  • a.min() 返回数组中最小的元素 a.argmin() 返回数组中最小元素所在的索引
  • a.max() 返回数组中最大的元素 a.argmax() 返回数组中最大元素所在的索引
  • a.sum() 返回所有元素的总和
  • a.mean()和np.median(a) 会返回数据的平均值
  • a.var() 返回数据的方差
  • a.std() 返回数据的标准差
  • a.sum(axis=0) 将每一行中对应的数据相加 （axis代表第一个维度，即行）
  • a.sum(axis=1) 将每一列中对应的数据相加 （axis代表第二个维度，即列）
  • a[x,y] a[a<3] a[(a>3)&(a%2==0)] a[0,0:2] ax,y,z返回指定的元素

numpy的典型应用——图片处理

对于彩色的图片可以用三维数组来表示(一二维分别表示行列，第三维存储了像素点的红绿蓝分量)

使用pillow这个库在python中读取图片文件

from PIL import Image

im = Image.open(‘doge.jpg’)

im.show()

通过np.array(im)将图片转换成一个numpy数组

im = np.array(im)

im.shape

  • im[100,100] 可以通过下标来访问某个像素点的颜色
  • im_r = im[:,:,0] 提取出所有像素点的红色分量
  • im_blend = im10.4 + im20.6 将两张图片按比例混在一起

注意：为了展示图片要把数组转换成整数

  • 可以利用跨度对图片进行采样
  • 跨度为-1，翻转图片

什么是机器学习框架？

是为了简化机器学习模型开发、训练部署而设计的软件工具或库，无需从零开始编写底层代码。

为什么要用机器学习框架？

避免重复造轮子，框架中已经实现一些经典算法，提高效率

前期准备

jupyter：专门在浏览器端可以以代码区块的形式对python进行编程、运行和数据展示的一个工具

Anaconda：包含python相关包和一些实用工具的集合，例如，我们可以使用anaconda进行Python环境的管理，使得在运行不同的程序时候使用不同的环境，极大避免了一些环境的依赖冲突

Miniconda：是一款轻量级的 Python 环境管理工具，是 Anaconda 的简化版本。它专注于提供核心的包管理和环境隔离功能，适合需要灵活控制依赖项的用户。(本人安装的是Miniconda)

conda：是Anaconda中的包管理工具

pytorch：是一个开源的深度学习框架，像NumPy 一样，PyTorch 也提供了多维数组（张量）和大量的数学运算。

基于YOLO的目标检测实验

一、使用yolov5算法进行图片、视频和摄像头实时内容检测

配置yolov5的运行环境

  1. 下载yolov5算法

官方下载地址：https://github.com/ultralytics/yolov5

备用下载地址：https://gitee.com/monkeycc/yolov5

权重文件下载： https://github.com/ultralytics/yolov5/releases/tag/v6.0

  1. 新创建一个yolov5的虚拟环境（此环境使用pip作为包管理工具！！！！）
  1. 官网安装pytorch（pip!!!）

  • 检验是否安装成功pytorch

  • 检验torchvision的版本号是否符合
  • 打开requirements.txt文件，将安装torch和torchvision的语句注释掉，将所做的修改进行保存

  1. 安装yolov5的配置环境，然后进入yolov5文件的终端启用yolo环境

  • 安装成功

运行yolov5算法

在终端输入：
（1） 检测摄像头内容：在cmd命令窗口输 python detect.py -–source 0 (0表示前置摄像头，1表示后置摄像头)

（2） 检测图片：将需要检测的图片放yolov5根目录，比如文件名为img.jpg，则输入命令 python detect.py -–source img.jpg

（3） 检测视频：随便找个视频，或者用手机录个短视频，放入根目录中，假设文件名问vid.mp4,则输入命令 python detect.py -–source vid.mp4

（4） 调用手机摄像头： 手机上下载IP摄像头 ，然后打开ip摄像头查看地址，然后输入以下命令

python detect.py –source http://admin:admin@192.168.101.6:8081

所有检测结果都保存在runs文件夹的detect文件中，所检测的照片和视频都放在根目录下

注意：这里需要将手机和电脑都连接到手机热点（保证两个设备都在同一局域网中）
